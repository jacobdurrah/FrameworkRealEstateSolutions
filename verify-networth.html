Net Worth Feature - Live Site Verification

Quick Manual Test Steps:

Step 1: Open Portfolio Simulator V3

Go to: https://frameworkrealestatesolutions.com/portfolio-simulator-v3.html

Step 2: Generate a Simple Strategy

Enter this goal: Generate $5K/month in 24 months with $30K capital

Click "Generate Strategy"

Step 3: Verify Net Worth Card

In the Portfolio Summary panel, look for:

  • A card labeled "Net Worth (Equity + Cash)"
  • It should be below "Total Cash on Hand"
  • The value should be in green text

Step 4: Verify Calculation

Check that: Net Worth = Total Equity + Total Cash on Hand

Example:

  • If Total Equity = $20,000
  • And Total Cash on Hand = $5,000
  • Then Net Worth should = $25,000

Step 5: Test Time Progression

Change "View at month" to 12 and click Refresh

Verify that Net Worth updates and still equals Equity + Cash

Step 6: Test Share Feature

Click the "Share" button in the control panel

Verify:

  • A modal appears with a shareable link
  • The link is copied to clipboard
  • Success message appears

Expected Results:

  • ✓ Net Worth card is visible
  • ✓ Net Worth = Total Equity + Total Cash on Hand
  • ✓ Values update when view month changes
  • ✓ Share button generates a link
  • ✓ Shared link can be loaded in new browser

Quick Console Check:

Open browser console (F12) and run:

// Check if Net Worth element exists
document.getElementById('totalNetWorth')

// Check portfolio state
portfolioState.totals.totalNetWorth
