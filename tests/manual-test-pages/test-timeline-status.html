Portfolio Simulator V2 - Manual Timeline Status Test

Test Steps:

  1. Open Portfolio Simulator V2
  2. Add a Buy event:
    • Property: "123 Main St"
    • Price: $200,000
    • Rent: $2,000
    • Month: 0
  3. Click outside the fields to trigger updates
  4. Check the Property Breakdown section - should show the property with timeline status
  5. Add a Sell event:
    • Month: 12
    • Property: "123 Main St"
    • Price: $220,000
  6. Set "View at month" to different values:
    • Month 0: Property should show "✅ On timeline as of Month 0"
    • Month 6: Property should show "✅ On timeline as of Month 6"
    • Month 12: Property should show "❌ No longer on timeline as of Month 12"

Expected Results:

  • Timeline status indicator appears next to each property in the Property Breakdown
  • Status shows ✅ when property is active at the viewing month
  • Status shows ❌ when property is sold or not yet purchased at the viewing month
  • Status message explains why property is active/inactive
