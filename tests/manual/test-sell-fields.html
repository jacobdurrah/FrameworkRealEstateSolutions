Manual Test for Sell Event Field Behavior

Test Steps:

  1. Open Portfolio Simulator V2
  2. Click "Add Event" to add a new row
  3. Fill in as a Buy event:
    • Property: 123 Test St
    • Price: $100,000
    • Down %: 20%
    • Rate: 7%
    • Term: 30 years
  4. Verify loan fields are calculated (Down $20,000, Loan $80,000, Payment ~$532)
  5. Change Action to "Sell"
  6. ✅ Verify all loan fields reset to 0:
    • Down %: 0
    • Down $: $0
    • Loan: $0
    • Rate %: 0
    • Term: 0
    • Payment: $0
  7. ✅ Verify these fields are disabled (grayed out) for Sell events
  8. Change back to "Buy" and verify fields are editable again

Test Cash from Sales:

  1. Add a Buy event for a property at month 0
  2. Add a Sell event for the same property at month 6
  3. Change Summary Month to 7
  4. Click Refresh
  5. ✅ Verify "Cash from Sales" shows the proceeds
  6. ✅ Verify "Total Cash on Hand" includes the sale proceeds

Test with V3:

  1. Open Portfolio Simulator V3
  2. Generate a strategy with flips
  3. ✅ Verify all Sell events have loan fields at 0 and disabled
